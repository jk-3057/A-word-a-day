# lexi 🌿

A personalized vocabulary-learning web app that helps users expand their vocabulary through daily word discovery and AI-powered writing practice.

## Overview

lexi is designed around a simple idea:

**Learn one useful word every day, then use it in a sentence.**

Instead of focusing solely on memorization, lexi encourages active vocabulary development by helping users apply newly learned words in their own writing and receive detailed feedback.

---

## Features

### 📚 Daily Word of the Day

* Personalized word selection based on user ability
* Beginner, Intermediate, and Advanced vocabulary levels
* Definitions, pronunciation, synonyms, and example sentences

### 🎯 Placement & Calibration

* Initial vocabulary placement quiz
* Difficulty calibration to determine appropriate word level
* Adaptive learning experience

### ❤️ Personalized Recommendations

* Like or dislike daily words
* Learns vocabulary preferences over time
* Recommends words based on previous feedback

### 🔊 Pronunciation Support

* Built-in text-to-speech pronunciation
* Quick audio playback for every word

### ✍️ AI Sentence Review

Powered by Gemini 2.5 Flash Lite.

Users can:

* Write a sentence using the daily word
* Receive an overall score out of 10
* Get feedback on:

  * Target word usage
  * Grammar
  * Sentence structure
  * Writing mechanics
* View corrected sentences and stronger examples

### 🔥 Progress Tracking

* Daily streak counter
* Total words learned
* Review history
* Writing score tracking

---

## How It Works

1. Complete the placement quiz.
2. Calibrate word difficulty.
3. Receive a personalized word each day.
4. Learn the definition, pronunciation, and usage.
5. Like or dislike the word to improve recommendations.
6. Write a sentence using the word.
7. Receive AI-generated feedback and suggestions.

## Technology Stack

* HTML
* CSS
* JavaScript (Vanilla)
* DictionaryAPI
* Google Gemini 2.5 Flash Lite
* LocalStorage for persistence

---

## Data Storage

lexi stores progress locally in the browser using LocalStorage.

Stored data includes:

* Vocabulary level
* Daily word history
* Likes and dislikes
* Personalization profile
* Review scores
* Learning streaks

No user account is required.
